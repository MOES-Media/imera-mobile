
# Imera Mobile

## Tech Stack

**Client:** React-Native, Recoil.js, TamaGUI, Expo



## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Building & Deploying

To deploy this project run

